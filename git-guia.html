git init

Inicializa un nuevo repositorio Git en el directorio actual.

git init

Este comando crea un nuevo subdirectorio llamado .git que contiene todos los archivos necesarios del repositorio.

git clone

Clona un repositorio existente en un nuevo directorio.

git clone <url_del_repositorio>

Ejemplo:

git clone https://github.com/user/repo.git

git status

Muestra el estado del repositorio actual, incluyendo los cambios que no han sido confirmados.

git status

git add

Añade archivos al área de preparación (staging area).

git add <nombre_archivo>

Ejemplo:

git add archivo.txt

Para añadir todos los archivos:

git add .

git commit

Guarda los cambios en el repositorio local con un mensaje descriptivo.

git commit -m "mensaje del commit"

Ejemplo:

git commit -m "Añadir archivo de ejemplo"

git push

Sube los cambios confirmados al repositorio remoto.

git push origin <nombre_rama>

Ejemplo:

git push origin main

git pull

Actualiza el repositorio local con los cambios del repositorio remoto.

git pull

Este comando combina los comandos git fetch y git merge.

git branch

Lista todas las ramas locales en el repositorio actual.

git branch

Para crear una nueva rama:

git branch <nombre_rama>

Ejemplo:

git branch nueva-rama

git merge

Combina cambios de una rama en la rama actual.

git merge <nombre_rama>

Ejemplo:

git merge feature-branch

git checkout

Cambia a una rama existente o crea una nueva rama y cambia a ella.

git checkout <nombre_rama>

Ejemplo:

git checkout main

Para crear una nueva rama y cambiar a ella:

git checkout -b <nombre_rama_nueva>

Ejemplo:

git checkout -b nueva-rama
